# System Design Academy — Question Banks

Comprehensive question banks for all 14 modules of the System Design Academy curriculum.

## Structure

Each module file contains **20+ questions** spanning three difficulty tiers:

| Tier | Approx. Count | Icon | Description |
|------|---------------|------|-------------|
| Beginner | 8-9 | 🟢 | Core concepts, definitions, basic trade-offs |
| Intermediate | 6-7 | 🟡 | Design reasoning, common pitfalls, moderate complexity |
| Advanced | 6-7 | 🔴 | Multi-step FAANG-level reasoning, whiteboard exercises |

## Question Types

- **Multiple Choice** — Test factual knowledge and concept recall
- **Open-Ended** — Evaluate depth of understanding and articulation
- **Whiteboard** — Simulate live system design interview prompts (2–3 per file)
- **Calculation** — Practice capacity estimation and numerical reasoning
- **Debug** — Identify and fix flawed system designs

## Usage

```
# Self-study
Pick a module, cover the answers, attempt each question.

# Interview prep  
Focus on 🟡 and 🔴 questions. Time-box whiteboard prompts to 25 min.

# Teaching
Use 🟢 questions for warm-up, 🔴 questions for advanced discussion.
```

Each question references the corresponding `Docs/XX-module-name.md` and its `-advanced.md` counterpart.
